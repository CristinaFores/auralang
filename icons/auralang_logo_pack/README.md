# AuraLang — Logo pack v1

Pack preparado a partir del logo aprobado para la primera implementación de AuraLang.

## Qué contiene

### `/chrome-extension`
Iconos listos para Manifest V3:

- `icon16.png`
- `icon32.png`
- `icon48.png`
- `icon128.png`
- `manifest-icons-snippet.json`

### `/web`
Assets para web/PWA:

- `favicon.ico`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png`
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `site.webmanifest`

### `/logo`
Versiones generales:

- `logo-app-*`: icono principal con fondo degradado morado/azul y A blanca.
- `logo-card-*`: icono sobre tarjeta blanca con A degradada.
- `logo-symbol-gradient.png`: símbolo A aislado con fondo transparente.
- `logo-symbol-white.png`: símbolo A blanco aislado con fondo transparente.
- PNG en varios tamaños: 16, 32, 48, 64, 128, 180, 192, 256, 512, 1024.
- WebP en 512 y 1024.

### `/svg`
- `logo-app.svg`
- `logo-card.svg`

### `/presentation`
- `logo-app-presentation-1024.png`
- `logo-card-presentation-1024.png`
- `wordmark-light.png`
- `wordmark-dark.png`

## Recomendación de uso

Para la extensión de Chrome usa `logo-app`, porque se lee mejor en 16x16 y 32x32.
Para landing/presentaciones puedes usar `logo-card` o el `wordmark` según el fondo.

## Colores base

- Morado principal: `#C13AF5`
- Azul principal: `#2637FF`
- Dark UI base: `#0B1220`
- Blanco tarjeta: `#FFFFFF`
